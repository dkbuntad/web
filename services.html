Sosialisasi

Event ini bertujuan untuk memperkenalkan program-program Diktisaintek Berdampak kepada mahasiswa di Universitas Tadulako. Guna untuk meningkatkan jumlah Mahasiswa yang mendaftar program-program Kampus Berdampak

Bimbingan Teknis

Event ini bertujuan untuk menjelaskan hal teknis terkait suatu proses penting pada program Diktisaintek Berdampak kepada mahasiswa di Universitas Tadulako. Guna untuk menambah wawasan teknis Mahasiswa pendaftar program Kampus Berdampak

Coaching

Event ini bertujuan untuk menjelaskan prosedur pemberkasan dan pendaftaran secara menyeluruh pada program Diktisaintek Berdampak kepada mahasiswa di Universitas Tadulako. Guna untuk meminimalisir kesalahan pada seleksi administrasi Mahasiswa pendaftar program Kampus Berdampak

MBKM Clinic

Event ini bertujuan untuk membantu Mahasiswa yang kesulitan dalam pemberkasan, pendaftaran, dan perolehan Rekomendasi pada program Diktisaintek Berdampak baik di SITAMPAN maupun pada platform Kampus Berdampak. Guna untuk mempermudah Mahasiswa dalam mendaftar program Kampus Berdampak

Tips dan Trik

Event ini akan memberikan tips dan trik praktis kepada mahasiswa di Universitas Tadulako terkait berkas maupun proses pendaftaran Diktisaintek Berdampak

Sharing dan Report

Event ini akan menjadi forum bagi mahasiswa untuk berbagi pengalaman dalam pelaksanaan program Diktisaintek Berdampak.

Mentoring

Event ini bertujuan untuk mendampingi Mahasiswa di Universitas Tadulako pada tahap pra, proses, dan after program Diktisaintek Berdampak. Guna sebagai sumber informasi Mahasiswa dalam program Kampus Berdampak
